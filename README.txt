Instrucciones:

1° Ejecutar Setear_Clave.
2° Ejecutar Proyecto.
3° Realizar la conexion sobre los pines detallados al inicio del archivo Proyecto.
4° Conectar a una fuente y navegar.

Importante: 

.La clave que se setea al ejecutar Setear_Clave es "J10R".
.Tener encuenta que se tiene 3 intentos de error de ingreso de clave. Pasados estos intentos se bloqueara el dispositivo.
.Para desbloquear se tiene un pin que mientras se presione permanecera "bloqueado", despresionar el mismo (ver archivo Proyecto).
